- [Visual studio code](https://code.visualstudio.com/)
- [Visual studio code Prettier extension](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

```json title=".vscode/settings.json"
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true,
  "editor.tabSize": 2,
  "files.autoSave": "onFocusChange"
}
```

## [Your Very First Webpage!](https://www.udemy.com/course/design-and-develop-a-killer-website-with-html5-and-css3/learn/lecture/27511930#overview)

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>My first webpage</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>My name is John, and this is my very first webpage :D</p>
  </body>
</html>
```
